# 14. 所得控除・税額控除

「所得控除」画面で、確定申告書の「所得から差し引かれる金額」「税金の計算」を試算する。

**Language**: **日本語** | [English](14-income-deductions_en.md) | [繁體中文](14-income-deductions_zh-TW.md)

> **この章を読み終えると**
> - aoiko が扱う範囲（事業所得ベースの所得控除・税額控除の試算）と、扱わない範囲（複雑な個別事情が絡む項目）の境界がわかる
> - 家族構成・保険料・医療費・寄附金等を入力して、基礎控除〜復興特別所得税までの試算結果を確認できる
> - `.xtx` 出力（確定申告）に所得控除・税額控除を反映させる手順がわかる
>
> **前提**：事業所得（決算書側の PL）が確定していること。[06. レポート](06-reports.md)。

## 1. aoiko が扱う範囲

aoiko は本来「事業の損益のみ」を扱うツールだが、利用者の希望により、確定申告書 KOA020 第一表の「所得から差し引かれる金額」「税金の計算」（下表）まで試算対象に含めている。

| 区分 | 内容 |
|---|---|
| 他の所得 | 給与所得（給与所得控除を自動計算）・雑所得（公的年金等は確定額入力、その他雑所得は収入−必要経費） |
| 不動産所得（設定でオプトインした場合のみ） | 損益は仕訳（事業/不動産切替で記帳した分）から自動算出。事業的規模の自己申告・青色申告特別控除の事業所得との共有枠配分・土地等取得の負債利子額による損益通算制限を試算 |
| 所得控除 | 基礎控除・社会保険料控除・小規模企業共済等掛金控除・生命保険料控除・地震保険料控除・医療費控除・寄附金控除・障害者控除・寡婦/ひとり親控除・勤労学生控除・配偶者控除/配偶者特別控除・扶養控除・特定親族特別控除 |
| 税額控除 | 配当控除・住宅借入金等特別控除・政党等寄附金等特別控除・住宅耐震改修特別控除等・外国税額控除等・その他の税額控除・災害減免額 |
| 税額計算 | 課税される所得金額・所得税額（累進税率）・復興特別所得税額・源泉徴収税額を差し引いた納付/還付見込額 |

> **雑損控除・住宅ローン控除・外国税額控除等は確定額を直接入力する方式。** 罹災状況・住宅ローンの年末残高推移・外国納税証明等、本人の個別事情に強く依存し制度も複雑なため、aoiko では計算せず、国税庁の計算書や税理士が算定した確定額をそのまま転記する。同様に、配偶者控除・扶養控除・特定親族特別控除・住宅ローン控除・外国税額控除等の「区分」コード（続柄や住宅ローンの契約種別等を表すコード）は出力しないため、これらの控除を使う場合は e-Tax 上で区分コードを補完すること。
>
> **公的年金等に係る雑所得も確定額を直接入力する方式。** 年齢・年金額・他の所得の合計額の3軸で変わる速算表で改定頻度も高いため、aoiko では計算せず、国税庁の「公的年金等に係る雑所得の速算表」等で算出した確定額をそのまま入力する。
>
> **不動産所得の地代家賃・借入金利子・税理士等報酬の支払先内訳（KOA220/KOA130 の明細ページ）も確定額を直接入力する方式。** 貸家等の状況（物件明細）は [08. 減価償却 § 2-4](08-depreciation.md#2-4-不動産所得用の固定資産設定でオプトインした場合のみ) の固定資産登録から入力する。

## 2. 入力項目

ナビゲーション **「所得控除」** から入力する。

- **他の所得**（任意）：給与所得（源泉徴収票の支払金額・源泉徴収税額を入力）、雑所得（公的年金等は確定額、その他雑所得は収入・必要経費）、事業所得側の源泉徴収税額（確定額）
- **不動産所得**（設定でオプトインした場合のみ表示）：事業的規模の該当有無（自己申告）、土地等を取得するために要した負債の利子の額（確定額）、地代家賃・借入金利子・税理士等報酬の支払先内訳（各項目、複数登録可）
- **家族構成**：配偶者の有無・合計所得金額・年齢、扶養親族（氏名・年齢・合計所得金額・同居老親等）を人数分登録
- **社会保険料・生命保険料等**：社会保険料・小規模企業共済等掛金の支払額、生命保険料（新/旧制度・一般/介護医療/個人年金の区分ごと）、地震保険料・経過措置（旧長期損害保険料）
- **医療費・寄附金・雑損**：医療費の支払額と保険金等の補填額、寄附金の合計額（ふるさと納税含む）、雑損控除額（確定額）
- **本人の状況**：障害者・特別障害者・ひとり親・寡婦・勤労学生の該当有無
- **税額控除**：配当控除・住宅ローン控除・政党等寄附金等特別控除・住宅耐震改修特別控除等・外国税額控除・その他の税額控除・災害減免額（いずれも確定額を入力）

入力は年度ごとに保存される（前年度の入力は自動で引き継がれない）。

## 3. 試算結果

各控除額・所得控除合計・課税される所得金額・税額控除後の所得税額・復興特別所得税額・所得税及び復興特別所得税の額・源泉徴収税額・納付/還付見込額を画面下部に表示する。事業所得は決算書側の集計から自動算出し、この画面では編集できない。合計所得金額は事業所得に「他の所得」で入力した給与所得・雑所得、および不動産所得（損益通算できる分）を加算した値で、基礎控除の級距判定等にはこちらを使う。

> 基礎控除は令和7年・8年分限定の暫定加算（措法41条の16の2）を反映しており、合計所得金額132万円以下なら95万円、655万円超2,350万円以下なら58万円など、所得水準に応じて変わる（令和9年分以後は2,350万円以下一律58万円に統一予定）。
>
> 給与所得控除は令和8・9年分限定の最低保障額引上げ特例を反映しており、給与収入220万円以下なら一律74万円（令和10年分以後は本則69万円に戻る予定）。
>
> 不動産所得がある場合、青色申告特別控除（65万/55万/10万）は事業所得と不動産所得の単一の共有枠で配分される（不動産所得から先に控除、残りを事業所得から控除）。事業的規模でない不動産所得は単独では65万/55万の対象外（10万が上限）だが、事業所得が単独で65万等の資格を満たしていれば、その枠は不動産所得にも及ぶ。

## 4. `.xtx` 出力への反映

[10. `.xtx` 出力](10-xtx-export.md) の確定申告書 `.xtx` 生成時、この画面で保存済みの入力があれば、所得控除・税額控除の各欄が自動的に載る。未保存（未入力）の場合は従来どおり出力されず、e-Tax 上で利用者が補完する。不動産所得を入力している場合は、確定申告書（KOA020）に加えて不動産所得用の決算書・収支内訳書（青色：KOA220、白色：KOA130）も同じ送信データに併載される。
