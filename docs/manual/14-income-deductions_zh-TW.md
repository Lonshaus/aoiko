# 14. 所得控除・税額控除

在「所得控除」畫面試算確定申告書的「所得控除」「稅額計算」。

**Language**: [日本語](14-income-deductions.md) | [English](14-income-deductions_en.md) | **繁體中文**

> **讀完這章你能做到**

> - 輸入家庭成員・保險費・醫療費・捐款等資料，查看基礎扣除〜復興特別所得稅的試算結果

## 1. aoiko 涵蓋的範圍

aoiko 原本只處理「事業損益」，但依使用者需求，也將確定申告書 KOA020 第一表的「所得控除」「稅額計算」（下表）納入試算範圍。

| 區分 | 內容 |
|---|---|
| 所得控除 | 基礎扣除・社會保險費扣除・小規模企業共濟等提撥金扣除・生命保險費扣除・地震保險費扣除・醫療費扣除・捐款扣除・身心障礙者扣除・寡婦/單親扣除・勤勞學生扣除・配偶者扣除/配偶者特別扣除・扶養扣除・特定親屬特別扣除 |
| 稅額扣抵 | 股利扣抵・住宅借款等特別扣抵・政黨等捐款特別扣抵・住宅耐震改修特別扣抵等・外國稅額扣抵等・其他稅額扣抵・災害減免額 |
| 稅額計算 | 課稅所得金額・所得稅額（累進稅率）・復興特別所得稅額 |

> **雜損扣除・住宅貸款減稅・外國稅額扣抵等採直接輸入確定金額的方式。** 因為罹災狀況・住宅貸款年末餘額推移・外國納稅證明等，強烈依賴本人的個別狀況且制度複雜，aoiko 不進行計算，直接轉錄國稅廳計算表或稅理士算出的確定金額。同樣地，配偶者扣除・扶養扣除・特定親屬特別扣除・住宅貸款減稅・外國稅額扣抵等的「區分」代碼（表示親屬關係或住宅貸款契約種類等的代碼）不會輸出，若要使用這些扣除，請在 e-Tax 上自行補齊區分代碼。

## 2. 輸入項目

從導覽列的 **「所得控除」** 進入輸入畫面。

- **家庭成員**：是否有配偶・合計所得金額・年齡，扶養親屬（姓名・年齡・合計所得金額・是否同居老親等）可依人數登錄
- **社會保險費・生命保險費等**：社會保險費・小規模企業共濟等提撥金的支付額，生命保險費（依新/舊制度・一般/介護醫療/個人年金區分），地震保險費・過渡措施（舊長期損害保險費）
- **醫療費・捐款・雜損**：醫療費支付額與保險給付等補償額，捐款總額（含故鄉稅），雜損扣除額（確定金額）
- **本人狀況**：是否符合身心障礙者・特別身心障礙者・單親・寡婦・勤勞學生
- **稅額扣抵**：股利扣抵・住宅貸款減稅・政黨等捐款特別扣抵・住宅耐震改修特別扣抵等・外國稅額扣抵・其他稅額扣抵・災害減免額（皆輸入確定金額）

輸入資料依年度儲存（不會自動沿用前一年度的輸入）。

## 4. 反映到 `.xtx` 輸出

在 [10. `.xtx` 輸出](10-xtx-export_zh-TW.md) 產生確定申告書 `.xtx` 時，若此畫面已儲存輸入資料，所得控除・稅額扣抵的各欄位會自動載入。若尚未儲存（未輸入），則維持不輸出，由使用者在 e-Tax 上自行補齊。
