# 15. 請求書・見積書の発行

「請求書・見積書」画面で、取引先への請求書・見積書を作成・発行する。

**Language**: **日本語** | [English](15-invoices_en.md) | [繁體中文](15-invoices_zh-TW.md)

> **この章を読み終えると**
> - 請求書・見積書を作成し、発行できる
> - 発行時に仕訳・売掛金が自動生成される仕組みがわかる
> - 発行済み文書の訂正方法（打消し仕訳）がわかる
> - 見積書から請求書へのワンクリック変換ができる
>
> **前提**：[01. 初回設定](01-setup.md) 済み。取引先（顧客）は事前に設定 → 「取引先」で登録しておく。

## 1. 請求書と見積書の違い

| | 請求書 | 見積書 |
|---|---|---|
| 番号系列 | 独立（例：INV-2026-0001） | 独立（例：QUO-2026-0001） |
| 発行時の会計処理 | 仕訳（借方売掛金・貸方売上高）＋ 売掛金（ArApEntry）を自動生成 | なし（成立前の提案のため） |
| 訂正方法 | 打消し仕訳（[12. 修正申告](12-amended.md) と同じ思想） | 発行後の内容変更はできない（取消のみ） |

見積書はあくまで「まだ成立していない提案」であるため、発行しても帳簿には一切影響しない。実際に受注が確定したら、見積書から請求書へワンクリックで変換する（§ 4）。

## 2. 請求書・見積書を作成する

1. ナビゲーション **「請求書」** から「請求書」または「見積書」タブを選択
2. **「新規作成」** をクリック
3. 宛先（取引先）・取引年月日・（請求書のみ）支払期限を入力
4. 明細行に品名・数量・単価（税抜）・税率を入力。**「＋ 明細を追加」** で行を増やせる
5. 消費税額は税率グループごとに1回だけ計算される（インボイス制度の「税率ごとに端数処理は1回」の原則に従う。明細行単位では計算しない）
6. 内容を確認して **「下書き保存」**（後で編集を続ける）または **「発行する」**

> **宛先の住所を請求書に載せたい場合**：設定 → 「取引先」で該当の取引先に送付先住所を登録しておく（任意項目）。

## 3. 発行すると何が起きるか

**請求書を発行すると：**

- 番号が確定する（設定で指定したプレフィックス + 年 + 連番、例：`INV-2026-0001`）
- 仕訳が自動生成される：借方 売掛金（税込合計）・貸方 売上高（税率グループ別、税込金額）
- 売掛金（ArApEntry）が生成され、[06. レポート § 10-2](06-reports.md#10-2-売掛金買掛金と現金流予測) の入金予定に反映される
- 内容がロックされる（下書きには戻せない）

**見積書を発行すると：**

- 番号が確定するのみ。仕訳・売掛金は一切生成されない

## 4. 見積書 → 請求書への変換

見積書の一覧から **「請求書に変換」** をクリックすると、明細をコピーした請求書の下書きが新規作成される（取引年月日は今日の日付、番号は未確定）。内容を確認・調整してから発行する。

## 5. 印刷して送付する

発行済みの文書は一覧から **「印刷」** をクリックするとブラウザの印刷ダイアログが開く。PDF として保存するには、印刷ダイアログの出力先で「PDFに保存」を選択する。

印刷レイアウトには、適格請求書等保存方式で必要な記載事項が自動的に含まれる：

- 発行者の名称・登録番号（設定 → 「基本情報」の事業者情報。免税事業者（登録番号未登録）の場合は登録番号欄が表示されない）
- 取引年月日
- 取引内容（明細）
- 税率ごとに区分した合計額及び適用税率
- 税率ごとに区分した消費税額
- 書類の交付を受ける事業者の名称（宛先）

> aoiko は PDF 生成や送付（メール送信等）の機能を持たない。ブラウザの印刷機能で作成した PDF を、利用者自身の方法（メール添付・郵送等）で送付する。

## 6. 発行済み請求書の訂正・取消

発行済みの請求書に誤りがあった場合、内容を直接編集することはできない（確定仕訳不変の原則、[12. 修正申告](12-amended.md) と同じ思想）。

1. 請求書の一覧から **「取消」** をクリック
2. 打消し仕訳（原仕訳の借方/貸方を入れ替えた仕訳）が生成され、元の仕訳は取消済みとして記録が残る
3. 売掛金（ArApEntry）も削除される
4. 正しい内容で新しい請求書を作成・発行する

> **入金記録がある請求書は取消できない。** 消込整合性が崩れるため、先に入金記録（[06. レポート § 10-2](06-reports.md#10-2-売掛金買掛金と現金流予測)）を取り消してから請求書を取消すこと。
>
> 見積書は仕訳を持たないため、内容の誤りは取消して作り直すだけでよい。
