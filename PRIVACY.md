# プライバシーポリシー

aoiko は **サーバを持たない** 純フロントエンドアプリです。利用者のデータは原則として **端末から外へ出ません**。本書では取得・送信されるデータの内訳を明示します。

## 収集する情報：なし

開発者・配布者は、利用者から以下のいずれも **取得しません**：

- 個人を特定できる情報（氏名・住所・電話・メール）
- 帳簿データ、仕訳、取引履歴
- API キー、認証情報
- 端末情報、IP アドレス
- 利用状況の解析（テレメトリ・アナリティクス）
- クッキー、ローカルストレージへのトラッキング

ホスティング先（公開時に GitHub Pages / Vercel / Cloudflare Pages 等を想定）の **HTTP アクセスログ** は、各サービスのポリシーに従って残る可能性があります。これは aoiko 側で制御できません。

## 端末内に保存されるデータ

ブラウザの **IndexedDB**（`aoiko` データベース）に以下を保存します：

| データ | 保存場所 | 送信先 |
|--------|----------|--------|
| 仕訳、明細、取引先、補助科目 | IndexedDB | 送信なし |
| 固定資産、家事按分ルール | IndexedDB | 送信なし |
| 申告済みスナップショット | IndexedDB | 送信なし |
| Gemini API キー | IndexedDB | 利用者が起動した LLM/OCR 機能の際のみ、Gemini API に送信 |
| 事業者情報（屋号、インボイス番号） | IndexedDB | 送信なし |
| バックアップフォルダのハンドル | IndexedDB | 送信なし |
| インポート履歴（ファイルハッシュ） | IndexedDB | 送信なし |

すべて利用者の端末ローカルにのみ存在します。ブラウザのサイトデータ削除を行うと **完全に消失** します。

## 端末外へ送信されるデータ

### OCR / LLM エンジン（利用者の任意・BYOK・設定で選択）

利用者が LLM 分類・領収書 OCR を **明示的に起動** した場合のみ、選択中のエンジンへ送信されます。送信内容：LLM 分類＝CSV 行テキスト（金額・摘要等）＋勘定科目一覧／OCR＝領収書画像（Base64）＋抽出指示プロンプト。

| エンジン（設定で選択） | 送信先 | 端末外送信 |
|------|------|------|
| Google Gemini（既定） | `generativelanguage.googleapis.com` | あり（クラウド） |
| OpenAI 互換 / Ollama 等：localhost 指定時 | 端末内（例 `http://localhost:11434`） | **なし** |
| OpenAI 互換 / Ollama 等：リモート指定時 | 利用者が指定したホスト | あり |
| Tesseract（純ローカル WASM OCR） | 画像は端末外に出ない。`jpn.traineddata` / `eng.traineddata` のみ初回 CDN から取得（自己ホスト可で完全オフライン） | **画像は無し**（traineddata DL のみ） |

- 送信は利用者のブラウザから **直接** 行われ、aoiko の管理サーバ（存在しない）は経由しません
- **外部（クラウド）送信エンジン使用時は送信直前に確認ダイアログ** を表示します
- Gemini：送信内容は Google のプライバシーポリシーおよび利用者の API プラン契約に従い、学習利用の有無は契約形態（無料枠 vs 有料枠）で異なります
- ローカル（Ollama 等を localhost で）使用時はデータが端末外に出ないため、最もプライバシー保護的です（OCR には vision モデル必須）
- Tesseract：LLM を使わず WASM OCR テキスト → 確定性ベース抽出（T+13 桁登録番号・日付・合計のみ）。精度は限定的で、店名・品目は推定しません。利用者が必ず人手で確認してください

### バックアップ（利用者の選択）

| 方法 | 送信先 |
|------|--------|
| File System Access API（Chromium） | 利用者が選択した **ローカル** フォルダ |
| OPFS（Safari / Firefox） | ブラウザ管理の **端末ローカル** ストレージ |
| 手動 JSON ダウンロード | 利用者の「ダウンロード」フォルダ |
| Google Drive 同期（将来実装） | 利用者の Google Drive |

aoiko 側のサーバには **一切** 送信されません。

## クッキー・トラッキング

- aoiko 自身はクッキーを使用しません
- サードパーティ広告・解析タグはありません
- リファラ送信はブラウザのデフォルト挙動に依存します

## 法的整合性

- 日本国内利用を想定しています。**個人情報保護法**に基づく「個人情報取扱事業者」要件は、aoiko の開発者・配布者が利用者から個人情報を取得しないため適用外と考えます。
- 利用者自身が aoiko を介して取り扱う他者の個人情報（取引先名・領収書の顧客名等）の管理責任は利用者にあります。
- EU 圏での利用は想定していませんが、GDPR 観点でも収集データなしのためデータ管理者該当性は低いと考えます。

## 変更履歴

本ポリシーは予告なく変更される可能性があります。重要な変更は GitHub の commit log で確認できます。
